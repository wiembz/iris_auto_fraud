# VHS — Mapping des libellés métier

Ce document décrit la traduction des codes techniques VHS en libellés compréhensibles par les utilisateurs métier de BNA Assurances.

---

## Niveaux d'attention proposés

| Code technique | Libellé métier |
|---|---|
| OK | État satisfaisant |
| DEGRADE | État à surveiller |
| IMMOBILISE | Usage déconseillé |
| CRITIQUE | Examen prioritaire suggéré |

---

## Lectures métier des points de contrôle

| Code technique | Libellé métier |
|---|---|
| OK | Élément conforme |
| WORN | Usure observée |
| WORN_STRONG | Intervention conseillée |
| BROKEN | Défaut confirmé |
| REPAIRED | Élément réparé |
| UNKNOWN | Information non exploitable |

---

## Niveaux d'état technique

| Niveau | Libellé métier |
|---|---|
| A | Aucun signal technique majeur |
| B | Quelques points à surveiller |
| C | Dégradation technique notable |
| D | Situation technique sensible |

---

## Remplacements recommandés dans les écrans IRIS

| Terme technique | Terme métier recommandé |
|---|---|
| score | Indicateur d'état |
| grade | Niveau d'état technique |
| checkpoint | Point de contrôle |
| penalty | Impact sur l'état du véhicule |
| règle de limitation du niveau final | Limitation métier du niveau final |
| normalized status | Lecture standardisée de l'observation |
| is_immobilizing | Point sensible pour l'usage du véhicule |
| WORN_STRONG | Intervention conseillée |
| BROKEN | Défaut confirmé |

---

**Les éléments présentés constituent une aide à l'analyse. La décision finale reste sous la responsabilité du gestionnaire.**
